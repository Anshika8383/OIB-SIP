# 🌡️ Temperature Converter

A simple and responsive **Temperature Converter** built using **HTML, CSS, and JavaScript**. This web application allows users to convert temperatures between **Celsius (°C)**, **Fahrenheit (°F)**, and **Kelvin (K)** instantly with an easy-to-use interface.

## 🚀 Features

* Convert Celsius to Fahrenheit and Kelvin
* Convert Fahrenheit to Celsius and Kelvin
* Convert Kelvin to Celsius and Fahrenheit
* Instant conversion
* Responsive design
* User-friendly interface
* Input validation
* Clean and organized code

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📖 How It Works

1. Enter the temperature value.
2. Select the input temperature unit.
3. Click the **Convert** button.
4. The converted values are displayed instantly in the other temperature units.
