# 🌐 Personal Portfolio Website

## 📌 Overview

This is my Personal Portfolio Website built using **HTML, CSS, and JavaScript**. The portfolio showcases my skills, projects, education, and contact information in a clean, responsive, and modern design. It serves as my online resume and helps recruiters and employers learn more about me.

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Modern and attractive user interface
* Home section with introduction
* About Me section
* Skills section with hover effects
* Projects section
* Contact section
* Smooth scrolling navigation
* Social media links
* Interactive animations using JavaScript

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
